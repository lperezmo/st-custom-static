# st-custom-static
Rebuilding streamlit repo with custom animation. Drop-in replacements to static folders (just find right version &amp; copy on top of /site-packages/static/ folder)

By default this is built using animation #3 (three dots), replacing the classic 'running man' on the official distribution.

### Available Animations

| # | Name | Preview |
|---|------|---------|
| 1 | Spinning Sync | ![Spinning Sync](example_animations/spin-sync.svg) |
| 2 | Pulsing Dot | ![Pulsing Dot](example_animations/pulse-dot.svg) |
| 3 | Three Dots | ![Three Dots](example_animations/three-dots.svg) |
| 4 | Bar Chart | ![Bar Chart](example_animations/bar-chart.svg) |
| 5 | Rotating Gear | ![Rotating Gear](example_animations/rotate-gear.svg) |
| 6 | Hourglass | ![Hourglass](example_animations/hourglass.svg) |
| 7 | Progress Ring | ![Progress Ring](example_animations/progress-ring.svg) |

If you want to built it yourself, detailed instructions are described on the `optional-build-yourself` folder.
